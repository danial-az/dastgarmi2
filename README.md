# DrawingApp - اپلیکیشن نقاشی ساده با React

یک اپلیکیشن ساده برای کشیدن اشکال هندسی (مربع، دایره، مثلث) روی یک بوم نقاشی با امکان درگ‌کردن شکل‌ها، حذف با دوبار کلیک، و قابلیت صادر و وارد کردن نقاشی به/از فرمت JSON.

---

## ویژگی‌ها

- انتخاب و درگ/دراپ اشکال هندسی (مربع، دایره، مثلث) از سایدبار به بوم نقاشی
- قابلیت تغییر موقعیت اشکال روی بوم (در نسخه فعلی فقط افزودن اشکال جدید امکان‌پذیر است)
- حذف شکل‌ها با دوبار کلیک روی آن‌ها
- نمایش شمارش هر نوع شکل ثبت شده روی بوم
- امکان خروجی گرفتن (Export) نقاشی به فایل JSON
- امکان وارد کردن (Import) نقاشی از فایل JSON برای ادامه ویرایش

---

## نحوه استفاده

### شروع

ابتدا پروژه را با دستورات زیر درون محیط React نصب و اجرا کنید:

```bash  
npx create-react-app drawing-app  
cd drawing-app  
# سپس فایل DrawingApp.js را به پروژه اضافه کنید و کد را در آن قرار دهید  
npm start  
رابط کاربری
سایدبار سمت چپ: اشکال مربع، دایره و مثلث که می‌توانید آن‌ها را انتخاب و با کشیدن روی بوم رها کنید.
بوم نقاشی: محل نمایش اشکال کشیده شده. هر شکل روی بوم با موقعیت دقیق قرار می‌گیرد.
حذف شکل: با دوبار کلیک روی هر شکل، آن شکل از بوم حذف می‌شود.
شمارش اشکال: تعداد هر نوع شکل در فوتر پایین صفحه نشان داده می‌شود.
دکمه Export: ذخیره ساختار اشکال روی بوم در یک فایل JSON دانلود شده.
دکمه Import: بارگذاری یک فایل JSON شامل شکل‌ها و نمایش مجدد آنها روی بوم.
نحوه کار برنامه
شکل‌ها قبلاً در سایدبار به صورت draggable تعریف شده‌اند.
هنگامی که کاربر یک شکل را از سایدبار می‌کشد و روی بوم رها می‌کند، موقعیت طول و عرض نسبی نسبت به بوم ثبت شده و شکل به آرایه shapesOnCanvas افزوده می‌شود.
هر شکل روی بوم به صورت مطلق (absolute) نمایش داده می‌شود و موقعیت آن با مختصات ذخیره شده تنظیم می‌شود.
حذف فرم‌ها با استفاده از فیلتر کردن آرایه در صورت دوبار کلیک روی هر شکل انجام می‌شود.
عملیات خروجی گرفتن و وارد کردن از طریق JSON به کمک JSON.stringify و FileReader پیاده‌سازی شده است.
ساختار کد
shapeTypes: آرایه‌ای شامل انواع اشکال پشتیبانی شده (square, circle, triangle)
shapeStyles: سبک‌های CSS مربوط به هر شکل
متغیر‌های state:
selectedShape: نوع شکل انتخاب شده (در این نسخه فقط برای درگ استفاده می‌شود)
shapesOnCanvas: آرایه‌ی اشکال موجود روی بوم با مشخصات id, type, x, y
توابع اصلی:
onDragStart: شروع کشیدن شکل و تنظیم داده انتقال
onDrop: افزودن شکل به بوم با موقعیت دقیق
onDragOver: اجازه دادن به درگ روی بوم
removeShape: حذف یک شکل بر اساس شناسه
exportToJson: خروجی گرفتن از آرایه اشکال به فرمت JSON
importFromJson: بارگذاری آرایه اشکال از فایل JSON
